Para hacerlo mas vistoso vamos a hacer nuestra serpiente a partir de una imagen. Para ello declaramos una nueva variable de tipo Image. 

				
					var ibody=new Image();

Tendremos que indicar la ruta donde tenemos esta imagen:

				
					ibody.src='images/body.png';

Por último, en nuestra funcion paint usaremos el método DrawImage(variable,x,y)

				
					ctx.drawImage(ibody,10,10);
