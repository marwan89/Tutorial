Una vez que tenemos nuestra imagen vamos a hacer que esta se mueva. Para ello, primeramente declaramos 2 variables que serán las coordenadas, es decir: la x y la y. 

			
				var x=50,y=50;

Declaramos una nueva función (act) que será llamada continuamente, por lo que la llamaremos en nuestra función previa run(). 

			
function act(){
    y+=2;
}
function run(){
    requestAnimationFrame(run);
    act();
    paint(ctx);
}

Y ahora cambiaremos nuestra linea 

				
					ctx.drawImage(ibody,10,10);

Por la siguiente: que lo que hace es sustituir los valores 10,10 que les dimos fijos por x e y, con lo que y irá aumentando como le hemos indicado en la función act() y nuestra imagen se desplazará verticalmente de arriba abajo.

			
				ctx.drawImage(ibody,x,y);

Evidentemente, la imagen, cuando tenga una y mayor que el alto de nuestro canvas desaparecerá por lo que vamos a controlar esto haciendo que nuestra imagen vuelva a aparecer arriba cuando esto pase. 

			
function act(){
    y+=2;
    if(y>canvas.height)
        y=0;
}
