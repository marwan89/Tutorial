Parte 2. Pintando nuestro personage.

En este punto vamos a dibujar nuestro personage dentro del canvas.
Para ello vamos a usar un requestAnimationFrame que es una función nueva que equivaldría a llamar un setTimeout(run,17), pero de forma optimizada. Para más información puedes visitar el siguiente enlace
En la función init() sustituimos la llamada que hacíamos a paint( ) por una nueva, run() que a su vez llamará a la función paint() y al requestAnimationFrame. 

			
function run(){
    requestAnimationFrame(run); 
    paint(ctx);
}

Ahora en la función paint() dibujamos nuestro personaje, inicialmente dibujaremos un cuadrado pequeño para ver cómo queda. 

				
function paint(ctx){
    ctx.fillStyle='#000';
    ctx.fillRect(0,0,canvas.width,canvas.height);
    ctx.fillStyle='#DF0101';
    ctx.fillRect(x,y,10,10);
}
